## ðŸŽ¬Getting Started

### Step 1. Clone this Repository

```
git clone https://github.com/CaesiumY/reactnd-myreads.git
```

### Step 2. Get inside!

```
cd reactnd-myreads
```

### Step 3. Install required packages

```
npm install
```

### Step 4. Run on Local Server

```

```
